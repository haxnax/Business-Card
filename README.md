# 💼 Business Card
<img width="347" height="568" alt="image" src="https://github.com/user-attachments/assets/4a26d6b8-898f-44d3-bc24-8b17365723cb" />

A modern, responsive digital business card created with HTML and CSS. This project showcases a professional business card design.

## 🎯 Features

- **Responsive Design** - Clean and centered layout that works on all devices
- **Professional Styling** - Modern color scheme with aqua background and rosybrown accent
- **Contact Information** - Displays email, phone, and portfolio link
- **Social Media Links** - Quick access to Twitter, LinkedIn, and GitHub profiles
- **Minimalist Design** - Simple and elegant presentation

## 🚀 Quick Start

1. Clone or download this repository:
   ```bash
   git clone https://github.com/yourusername/business-card.git
   ```

2. Open the project folder and launch the HTML file:
   - Double-click `index.html` to open in your default browser
   - Or use a live server extension in VS Code

3. View the digital business card in your browser

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Styling and layout (with flexbox-friendly design)
- **Responsive Meta Tags** - Mobile-friendly viewport configuration

## 🎨 Customization

To customize this business card for your own use:

1. **Edit Personal Information** in `index.html`:
   - Update name, title, company, email, and phone number
   - Replace profile image URL
   - Update social media links

2. **Modify Styling** in `css/styles.css`:
   - Change background colors
   - Adjust card dimensions (width, padding)
   - Customize fonts and spacing

3. **Add Local Images**:
   - Place profile image in the `images/` folder
   - Update the `src` attribute to reference the local file
